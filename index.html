“我的BLOG部署问题汇总”

##第一章：

这是通过ssh生成的链接，所以配置时也需要更改为ssh类链接.

$: vim _config.yml ：

deploy:

type: git

repo: https://github.com/yourname/yourname.github.io.git

branch: main # 因为zzzq的关系，官方将默认的master改为main，注意.

其中的repo修改为

repo: git@github.com:yourname/yourname.github.io.git

注意：这里的链接没有填入个人网址.

参考文献：https://blog.zhheo.com/p/128998ac.html
